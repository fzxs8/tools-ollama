# PromptPilot界面设计

## 1. 整体布局

采用经典的三栏式布局设计：
- **左侧栏**: 模型选择和功能导航区域
- **中间主区域**: Prompt生成、展示和优化区域
- **右侧抽屉**: Prompt管理面板，可滑入滑出

## 2. 左侧栏设计

### 2.1 服务选择区域
- 显示已配置的Ollama服务列表
- 支持单选一个服务作为当前工作服务
- 显示服务基本信息（名称、地址等）

### 2.2 模型选择区域
- 显示选中服务下的可用模型列表
- 支持复选框选择多个模型
- 显示模型基本信息（名称、大小、修改时间等）
- 限制最多选择3个模型进行对比测试

### 2.3 功能导航
- 快速访问链接到主要功能区域
- 当前页面高亮显示

## 3. 中间主区域设计

### 3.1 想法输入区
- 大型多行文本输入框
- 占据页面上部显著位置
- 提供清晰的提示文字"请输入您的想法或需求描述"
- 支持长文本输入

### 3.2 操作按钮区
- 「生成Prompt」主按钮，醒目颜色
- 「优化」按钮
- 「保存」按钮
- 「已保存」按钮
- 按钮排列整齐，间距合理，位于右下角

### 3.3 Prompt展示区
- 流式内容展示区域
- 支持Markdown渲染
- 显示生成过程中的实时内容
- 提供内容复制功能
- 显示生成状态和进度

### 3.4 多模型对比区
- 分栏展示不同模型的输出结果
- 每栏显示模型名称标识
- 支持横向对比查看
- 提供选择最佳结果的功能

## 4. 抽屉设计

### 4.1 优化抽屉
- 从右侧滑出的表单抽屉
- 包含反馈输入区域
- 包含确认和取消按钮

### 4.2 保存抽屉
- 从右侧滑出的表单抽屉
- 包含名称、描述和标签输入项
- 支持创建新标签
- 包含确认和取消按钮

### 4.3 已保存抽屉
- 从右侧滑出的管理面板
- 展示所有已保存的Prompt列表
- 每个Prompt项包含预览和操作按钮
- 支持预览、编辑和删除操作

## 5. 交互设计规范

### 5.1 流式展示效果
- 文字逐字显示，模拟打字机效果
- 提供暂停/继续功能
- 支持加速/减速控制
- 显示生成进度指示器

### 5.2 按钮交互状态
- 默认状态：标准颜色和样式
- 悬停状态：颜色加深，提供视觉反馈
- 禁用状态：灰色显示，不可点击
- 加载状态：显示加载动画

### 5.3 服务和模型选择交互
- 服务选择后自动加载对应的模型列表
- 模型列表加载过程中显示加载状态
- 当用户尝试选择第4个模型时，显示明确的错误提示
- 超过限制的模型项显示为禁用状态
- 提供友好的限制说明文本

### 5.4 抽屉交互
- 所有抽屉从右侧滑入，提供流畅的动画效果
- 点击遮罩或关闭按钮可关闭抽屉
- 抽屉内容区域具有适当的内边距和布局

### 5.5 错误处理反馈
- 提供友好的错误提示信息
- 错误信息显示在相关操作区域附近
- 提供重试或解决建议

## 6. 视觉设计规范

### 6.1 色彩方案
- 主色调：延续应用整体风格的蓝色系
- 辅助色：绿色（成功）、橙色（警告）、红色（错误）
- 背景色：浅灰背景，确保内容清晰可读
- 文字色：深灰主文，浅灰辅助信息

### 6.2 字体规范
- 标题：较大字号，加粗显示
- 正文：适中字号，正常字重
- 辅助信息：较小字号，浅色显示
- 代码内容：等宽字体，特殊背景

### 6.3 间距规范
- 组件间保持适当间距
- 内容区域有足够的内边距
- 文本段落间有合适的行高
- 按钮间保持统一间距

## 7. 响应式设计

### 7.1 桌面端适配
- 充分利用大屏幕空间
- 保持三栏布局结构
- 字体和组件大小适中

### 7.2 平板端适配
- 左侧栏可折叠收起
- 主区域自适应宽度
- 抽屉全屏显示或适当缩小

### 7.3 手机端适配
- 单栏布局，各区域垂直排列
- 左侧栏和抽屉改为底部或顶部导航
- 按钮和输入框适当放大便于触控

## 8. 用户体验优化

### 8.1 操作引导
- 提供首次使用引导
- 关键功能添加提示说明
- 复杂操作提供步骤指引

### 8.2 快捷键支持
- Ctrl+Enter快速生成Prompt
- Ctrl+S保存Prompt
- ESC关闭抽屉等面板

### 8.3 状态同步
- 所有操作状态实时同步
- 网络异常时提供本地缓存
- 操作完成后给出明确反馈
